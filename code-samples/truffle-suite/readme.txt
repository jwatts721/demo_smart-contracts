Creating a Truffle project using an existing Truffle Box
    $ mkdir pet-shop-tutorial
    $ cd pet-shop-tutorial

    $ truffle unbox pet-shop



You can create an empty Truffle project
    $ truffle init


Compile, Deploy and Test Smart Contract
1 - truffle compile
2 - truffle migrate
3 - truffle test
